# Profile Checklist

Use this before making the account public-facing.

## Identity

- Username is professional
- Display name matches resume and LinkedIn
- Profile photo is professional
- Bio explains your throughline in one sentence
- Location and contact details are appropriate for public visibility

## Profile README

- First line states what you do clearly
- Summary connects software, GIS, databases, and analytics
- Focus areas are grouped by capability
- Links to LinkedIn or portfolio are added
- Pinned repos tell a coherent story

## Repository Quality

- Public repos have clear names
- READMEs explain problem, solution, and stack
- Sensitive data and secrets are removed
- Screenshots or outputs are included where useful
- Topics and descriptions are filled in

## Professional Signal

- Public work looks intentional, not accidental
- Repo count is curated, not noisy
- At least one repo shows end-to-end engineering
- Activity looks current or projects have clear completed status
