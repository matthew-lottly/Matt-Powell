# Public Repo Metadata Audit

Reviewed on 2026-04-02.

Scope
- Collaboration-ready showcase repos
- Newly published supporting repos
- Research-heavy repos carrying citation metadata

Audit summary
- All audited public-facing project folders include a `README.md` and `LICENSE`.
- Contributor guides now exist for the seven collaboration-ready showcase repos plus the research repos that already carried public contribution notes.
- Citation metadata is present in the current research-package repos where citation support is materially useful: `causal-lens` and `strata`.
- The remaining supporting repos without `CONTRIBUTING.md` are still intentionally treated as lower-collaboration or more experimental surfaces.

Current guidance
- Add `CONTRIBUTING.md` only where outside collaboration is realistic.
- Add `CITATION.cff` only where the repository has a publication, package, or research workflow that benefits from formal citation metadata.
- Re-audit metadata whenever a repo moves from reviewable to operational prototype or becomes a collaboration priority.
