# Publishing Notes

This project is intended to live as a standalone Python package under `projects/causal-lens` and later as its own public repository.

## Release Checklist

1. Run the test suite.
2. Build the package.
3. Validate distribution metadata.
4. Tag the release after the package and docs are in sync.

## Journal Targets

1. JOSS as the primary software-paper venue.
2. JSS as a later, more method-heavy software-statistics venue.
