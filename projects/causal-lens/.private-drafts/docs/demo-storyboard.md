# Demo Storyboard

## Story

1. Generate a synthetic observational dataset with confounding.
2. Estimate treatment assignment probabilities.
3. Compare regression-adjustment, matching, IPW, and doubly robust estimates.
4. Inspect overlap and covariate balance.
5. Inspect additive-bias sensitivity and subgroup treatment-effect summaries.
6. Export a concise JSON report for downstream review.

## Reviewer takeaways

1. The package has a consistent API.
2. Diagnostics are available without custom notebook code.
3. Multiple estimators can be compared against the same dataset and estimand.
4. Phase 2 summaries provide a first pass on robustness and treatment-effect heterogeneity.
